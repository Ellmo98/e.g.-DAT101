    1. Hvordan kan DOM hjelpe oss utvikle interaktive nettsider?

      2. DOM gjør det mulig for JavaScript å hente, endre og opprette elementer dynamisk. Dette gjør nettsider interaktive og reagerer på brukerhandlinger som tastetrykk, klikk og valg.

    2. Hva er et event? Forklar enkelt hva som skjer når vi legger til en event listener.

      3. Et event er en handling i nettleseren, som et klikk eller tastetrykk. En event listener kjører en funksjon hver gang den hendelsen skjer.

    3. Forklar event-objektet og hvordan vi kan hente data fra det.

      4. Event-objektet gir informasjon om hendelsen, som hvilken tast som ble trykket eller hvilken musposisjon som ble brukt. Vi kan bruke f.eks. event.key eller event.target for å hente data.

    4. Forklar hovedforskjellen mellom DOM id og DOM name. Kan vi ha identiske IDs?

      5. id er unik og kan kun brukes én gang per side. name kan brukes på flere elementer, og brukes ofte for grupper som radioknapper. Flere elementer kan ikke ha samme id.
